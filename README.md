# 우리 가족 안전 30일

어린이가 가족 구성원과 함께 태풍, 집중호우, 지진에 대비하며 30일을 보내는 2D 웹 교육 게임입니다. 기존 태풍 경로 예측 프로젝트는 참고만 했으며, 이 프로젝트는 `AX_Games` 안에서 독립적으로 실행됩니다.

## 실행 방법

Node.js 18 이상이 필요합니다.

```powershell
cd C:\kdh\AX_project\AX_Games
npm install
npm run dev
```

브라우저에서 `http://localhost:5174`로 접속합니다.

```powershell
npm test -- --run
npm run build
```

## 구현된 기능

- 엄마, 아빠, 할머니, 할아버지, 누나, 형, 본인 중 1~3명 선택
- 태풍, 집중호우, 지진 선택
- 선택 행동으로 진행하는 1~30일 생존 루프
- 매일 3분 남은 시간을 보여주는 안내 타이머(시간 만료 시 자동 진행 없음)
- 선택 직후 결과 스토리, 물품·점수·체력 변화 요약, 안전교육 해설
- 준비·돌봄·탐색·위험 행동에 따라 달라지는 캐릭터 모션
- 안전 선택 연속 실천에 따른 콤보 경험치와 누적 XP 기반 안전 등급
- 안전 첫걸음, 안전 습관, 준비 대장, 공동체 협력 등 디지털 배지
- 결과 경험 → 안전 원리 성찰 → 다음 날 전략 선택으로 이어지는 경험 학습 루프
- 다음 날 전략은 10개 전략 풀에서 매번 중복 없는 3개를 무작위로 제안
- 사전 대비·정보 확인·안전 대피·물품 관리·공동체 협력을 분석하는 개인화 학습 리포트
- 태풍+정전, 지진+쓰나미 등 복합 재난 사건 체인을 추가할 수 있는 시나리오 메타데이터
- 인원 비례 시작 물품: 1명당 빵 2개, 물 2개, 구급상자 1개
- 빵과 물을 소비하면 해당 가족의 배고픔과 갈증이 각각 30 회복
- 체력, 배고픔, 갈증과 상태별 표정
- 회복, 파밍, 힌트, 위험 예측, 방어, 학습 등 가족별 능력
- 재난별 일일 사건, 안전 선택지, 결과와 교육 팁
- 편의점, 약국 구조함, 학교 대피소, 구호 상자, 공공 급수대, 물웅덩이 등 6종의 무작위 파밍 사건
- 파밍으로 빵·물뿐 아니라 구급상자도 획득 가능
- 물품 획득을 시도하는 파밍은 성공 50%·미획득 50%로 판정되며, 성공 시 기본 빵 4개·물 3개가 그날 소비 후 비상 물품에 저장
- 재난과 위험도에 반응하는 코드 기반 2D 거실
- 정전, 창문, 구조 방송, 이웃, 침수, 선반 등 데이터 기반 사건
- 준비도와 구조 연결, 가족 상태에 따른 여러 엔딩
- 모바일 단일 열 레이아웃, 키보드 포커스, 모션 감소 설정

가족의 체력이 0이 되면 “행동 불가” 상태로 표시되고 구조를 기다립니다. 결과 화면 역시 어린이가 다시 안전 수칙을 시도해 볼 수 있는 따뜻한 표현을 사용합니다.

## 사건과 캐릭터 추가

새 사건은 `src/data/gameData.ts`의 `EVENTS` 또는 `FARM_EVENTS`에 추가합니다. `disasters`에 적용할 재난을 지정하고, 각 선택지에 결과 문장, 안전 팁, 상태 효과를 작성하면 게임 엔진이 자동으로 처리합니다.

새 캐릭터는 `FamilyId` 타입과 `FAMILY` 배열에 식별 정보, 색상, 소품, 능력을 추가합니다. 능력의 실제 계산은 `src/game/engine.ts`에 작은 보정 규칙으로 연결할 수 있습니다.

## 교육 포인트

- 강풍에는 창문과 거리를 두고 손으로 유리를 막지 않기
- 침수된 길과 맨홀 주변에 들어가지 않기
- 지진에는 엎드리고, 가리고, 붙잡기
- 정전에는 촛불보다 손전등 사용하기
- 공식 방송을 확인하고 가족과 이동 경로 공유하기
- 인원수에 맞춰 물, 식량, 구급용품 준비하기

## 해커톤 발표에서 강조할 점

1. 캐릭터가 장식이 아니라 선택 결과를 바꾸는 능력과 상태를 가집니다.
2. 사건 데이터와 게임 엔진을 분리해 교사가 안전교육 내용을 쉽게 확장할 수 있습니다.
3. 재난 위험도가 거실의 빗물, 균열, 흔들림으로 즉시 시각화됩니다.
4. 서버와 외부 API 없이 안정적으로 시연하며, 실제 타이머 대신 행동으로 날짜가 진행됩니다.
5. 좋지 않은 결과도 벌보다 “왜 안전하지 않았는지”를 알려주는 학습 기회로 표현합니다.

## 안전 성장 시스템

안전한 선택은 기본 25 XP를 제공하며 연속 실천 시 콤보 보너스가 더해집니다. 위험한 선택은 콤보를 초기화하지만, 결과 화면에서 안전 원리를 확인하고 다음 날 전략을 다시 선택할 수 있습니다.

각 선택은 학습 영역과 안전 품질로 기록됩니다. 30일 종료 후에는 영역별 달성도, 잘 지킨 수칙, 더 연습할 영역, 획득한 배지를 학습 리포트로 확인할 수 있습니다.

복합 재난 시나리오는 `COMPOUND_SCENARIOS`와 각 사건의 `scenario` 메타데이터를 사용합니다. 현재 구조는 교육용 시나리오 확장을 위한 것이며, 실제 재난 데이터를 연결할 때는 출처와 적용 지역을 `sourceNote`에 기록할 수 있습니다.
